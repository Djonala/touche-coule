Touché, Coulé

Faire un Touché-coulé

  1. Page de choix des speudo : enregstrer dans la BDD d'une partie + stocker l'id de la partie dans la session
  2. Page de placement des bâteaux : L'utilisateur ne fait pas d'erreur de saisie. Mettre à jour la partie. Exemple de coordonnée : C2, C7
  3. Page du joueur1 : Le joueur saisie les coordonnées d'une case et envoi (Ex. B5).
    • Si le coup est dans l'eau alors mettre le symbole 💧 (en html : &#128167;)
    • Si le coup touche un bateau alors mettre le symbole 💥 (en html : &#128165;)
    • Si le coup coule le dernier bateau alors mettre la page end.html, étape 5.
    • Dans tous les autres cas, c'est au joueur adverse de jouer.
    • Enregistrer tous les coups dans la BDD.
  4. Page du joueur2 : idem
  5. Page de fin : Indique la fin de la partie
